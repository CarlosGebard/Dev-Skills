# Dev Skills

Repositorio personal de skills y prompts usados en flujos de desarrollo.

## Contenido

- `planner.md`: guia para planificar trabajo tecnico.
- `low-token.md`: instrucciones para trabajar con bajo consumo de contexto.
- `Skills.md`: notas generales sobre skills.
- `Agents.md.md`: instrucciones para agentes.
- `Skills-Documentation/`: skills orientadas a documentacion tecnica.

## Uso

Estas notas estan pensadas para usarse desde el vault de Obsidian y versionarse en GitHub.
